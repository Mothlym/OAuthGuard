Guide

OAuthGuard's user interface is accessed with an icon in the Chrome web page and provides buttons that redirect a user to other pages with information or functionality.

The tabs "OAuthGuard", "Warnings" and "Tools" are available.

Under the "OAuthGuard" tab it shows how many attacks detected are categorized by the type of attack detected (The CSRF blocked, refers to the token leaks that were blocked, third-party token leaks blocked, and the number of HTTPS upgrades made).

The "Warnings" tab lists the websites affected by impersonation attacks.

Under the "Tools" tab there are five buttons, "Check Report", "Open OAuth 2.0 Tools", "Export Data", "Turn HTTPS Upgrade On / Off" and "Security Issues Description". If you select the report button it will list all deficiencies sorted by domain. Open OAuth 2.0 Tools opens a window where domains can be searched for to get detailed information inquiry / response information made for this domain. Exporting data creates a file with the data stored in localStorage that will be downloaded. Turning on / off HTTPS upgrade turns on / off the feature that tries to redirect traffic from HTTPS to HTTP. Vulnerability Description Description lists information about the security deficiencies OAuthGuard detects / protects against.
