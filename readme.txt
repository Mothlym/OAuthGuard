How to install OAuthGuard on Chrome.

1. download latest Chrome web browser.
2. use Chrome open chrome://extensions/ 
3. click the Load unpacked extension
4. select the extension directory. e.g. /somepathto/OAuthGuard/, click select button of the popup window.
5. enjoy OAuthGuard, you will find a small red icon on the right top corner of Chrome, click the icon to explore OAuthGuard
